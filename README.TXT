REQUIREMENT :
1. Apache / IIS Server
2. PHP 5.x.x
3. MySQL Server 5.x

Howto INSTALL :
1. put all this file in subfolder "pooling" on your htdocs root folder (public_html folder) with php & mysql support
2. create database on your MySQL server with name "pooling" (without quote)
3. import file /db/pooling.sql into your MySQL server database
4. edit file /application/config/database.php to suit with your MySQL connection (username, password, port, dbname)
5. access for front end with : http://localhost/pooling/ (assumtion you put on folder pooling)
6. access for backend (admin) : http://localhost/pooling/login
